# 认养一棵树小程序

This is a Vue 3 + Vite project for a tree adoption mini-program. It includes 7 core pages:
1. Home page with tree species selection
2. Tree adoption form page
3. My tree 3D growth page
4. Watering and fertilizing page
5. Offline orchard appointment page
6. Co-adoption invitation page
7. Fruit market page

## Project Setup

```bash
npm install
```

### Compile and Hot-Reload for Development

```bash
npm run dev
```

### Compile and Minify for Production

```bash
npm run build
```
